Rhino 8-Plugin [Update]-Convert Dimensions Between Dynamic And Static

In this updated version, I added a second command called StaticToDim that enables the user to convert/revert static dimensions back to dynamic. The first version only had one command, DimToStatic, that converts dynamic dimensions into static dimensions, which is useful in scenarios where you don't want the dimensions to update when scaling the scene objects. For example, in cases where you want to print out, and you want to scale the drawing to fit a paper size, etc. The reason for the second command is that you might save the file before undoing the scene, and the dimensions are still static, or if you receive a file with static dimensions.

What's New:

  • New StaticToDim command: revert static dimensions back to dynamic
  • Original DimToStatic command: make dynamic dimensions static for printing and for cases where you don’t want dimension values to update as you scale the drawing
  • Improved workflow for files with mixed/static dimensions
