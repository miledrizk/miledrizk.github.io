Grasshopper 3D - Parametric Cylindrical Lamp

In this tutorial, we will learn how to create a parametric cylindrical lamp shape with a variable number of sections that have bulges, and these bulges are not at the same level. In the end, we'll end up with a model where we can control almost all of its aspects parametrically. You'll learn many tools and tricks during this tutorial, so stay tuned until the end.

What You'll Learn:

  • Creating a parametric cylindrical lamp with variable sections
  • Working with bulges at different levels
  • Parametric control over all model aspects
  • Advanced Grasshopper tools and tricks

  • How to model a lamp body made of blades in Grasshopper 3D
  • Using spinners for parametric control of model aspects
  • Techniques for generating complex, customizable forms
  • Approaches for adapting this workflow to create other shapes
